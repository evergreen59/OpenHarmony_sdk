# Base64编解码

### 介绍

Base64是一种基于64个可打印字符来表示二进制数据的方法，本示例展示了Base64编码及解码的结果输出。

使用说明：
1. 点击**编码**按钮，将会把字符串转换为ASCII码，然后对ASCII码进行编码，并在文本框输出编码后的结果。
2. 点击**解码**按钮，将会把编码后输出的结果作为解码器的入参，解码后得到ASCII码数组，然后将ASCII码数组转换为字符串在文本框输出。
