## Functions

This demo shows the functions of On Listner For Renderer State Changes,Capturere State Changes , 
Get Renderer State Change information (Client UID, Stream ID ,Renderer State , Renderer Flag, Content and Stream Usage)  
Get Capturer State Change information (Client UID, Stream ID ,Capturer State , Capturer Flag and Source)  
displaying information.

## Instructions

After the program runs, we need to send the sample wav  files to the device. The commands are as follows. (The first parameter
after "send" is the path of the local file)

```
hdc_std file send   .\PinkPanther60-44100-1c.wav  /data/app/el2/100/base/ohos.acts.multimedia.audio.audiomanager/haps/entry/cache

```
